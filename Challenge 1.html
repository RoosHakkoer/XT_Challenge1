De klok

3500 - 1500 voor Christus - heden

De wijzers van een klok die we vandaag nog steeds gebruiken gaan rechtsom. Dit komt doordat de richting van de zonnewijzer’s schaduw ook die kant op ging. De oudste zonnewijzers waren obelisken (3500 voor Christus) en schaduw klokken (1500 voor Christus). Een dag heeft 24 uur, met elk uur 60 minuten en die bestaan weer uit 60 seconden. Deze theorie is ook toegepast op een digitale klok. Deze digitale klok heeft iedereen wel op zijn smartphone staan, maar de analoge klok wordt nog steeds veel gebruikt. Van zonnewijzer naar analoge en digitale klok zijn de echo's van het verleden die in het heden verder leven.

CERN

1954

Cern is de European Organisation for Nuclear Research. Het CERN is een organisatie die onderzoek doet naar elementaire deeltjes met behulp van een deeltjesversneller. De grootste deeltjesversneller van het CERN is de LHC (Large Hadron Col lider) op de Frans-Zwitserse grens bij Genève.

World Wide Web

1989

De computer wetenschapper Tim Berners-Lee, die in dienst is bij CERN, is zelf een project begonnen voor het managen van informatie. Tim had een plan bedacht waardoor je digitaal informatie met elkaar kom uitwisselen. In eerste instantie het hij een software bedacht en noemde hij het ‘Enquire’ dat vernoemt is naar het boek: ’Enquire Within Upon Everything.’ Daarna gin hij verder om het groter aan te pakken, waaruit een document is ontstaan met de titel: ‘Information Management: A Proposal.’ Dit is het begin van het World Wide Web wat we nu kennen.

HyperText Transfer Protocol

1990

HTTP kan data verzenden en data ontvangen via het web. Dit gaat door middel van een unieke URL. De URL pagina’s zijn opgebouwd uit HTML. In de HTML code kan je linken van een URL naar een andere URL door het HREF attribuut te gebruiken in het A element. Deze links maken het web een hypertext system.

HyperText Markup Language

1990

De eerste versie van HTML had 21 elementen. Veel van deze elementen gebruiken we nog steeds in het HTML van nu. Tim Berners-Lee maakte des tijds gebruik van een NEXT computer dat ontworpen was door Steve Jobs. Hiermee heeft Tim HTTP,HTML en de eerst web browser mee gemaakt wat toen alleen nog op de NEXT machines werkte.

Cascading Style Sheets

1994

Håkon werkte ook bij CERN in de zelfde tijd als Tim Berners-Lee. Håkon zag veel mogelijkheden in het World Wide Web en de HTML-taal. Hij introduceerde Cascading Style Sheets. Met CSS kan je nu visuele elemeten aan je HTML pagina toevoegen. Er is nu heel veel visuele variatie waarmee je onderscheid tussen websites kan maken.

Webpages are not printed paper

1999

Volgens John Allsopp heb je bij het beginnen van een nieuw medium altijd te maken met kenmerken van een vorig medium. Daarom is het ook niet raar dat webdesign begon met het maken van webpagina’s die er uitzagen als print layouts. Designers konden hier goed mee werken, omdat ze bekend waren met dit soort layouts. Maar web is geen papier. Er is geen vaste hoogte en breedte in het webdesign, waardoor je veel meer vrijheid hebt in het bouwen van een webpagina.

iPhone

2007

Thuis surfen op het internet is voor iedereen normaal, maar wat als je ook buitens huis kan internetten. Steve Jobs introduceerde in 2007 de eerste iPhone. Deze telefoon past in je zak en je kan er mee internetten. Voordat de iPhone uitkwam waren er al wat mobile apparaten die webpagina’s konden weergeven, maar dat waren niet de webpagina’s die we op de PC konden zien. De iPhone daarentegen kon wel webpagina’s weergeven zoals dat van een computer. Dit zorgde voor een nieuwe ontwikkeling rond het webdesign.
